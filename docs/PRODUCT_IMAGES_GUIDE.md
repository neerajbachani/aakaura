## How to Add Actual Images

### Option 1: Add Your Own Product Photos

1. Take 4-5 photos of the Root Chakra Wall Hanging product
2. Save them in the `public/images/products/` directory with the names above
3. The gallery will automatically display them!

### Option 2: Use Placeholder Images

For now, you can create placeholder images or use stock photography:

1. Create the directory: `public/images/products/`
2. Add any product images with the names listed above
3. Recommended sizes: 1200x900px or similar (4:3 aspect ratio works best)
